# Superpowers

**Criador**: [obra](https://github.com/obra)
**Repositório Original**: https://github.com/obra/superpowers

## Descrição

Framework de orquestração multi-agente. Conjunto de skills para desenvolvimento orientado por agentes com verificação e planejamento sistemático.

## Skills Incluídas

- `brainstorming/` - Brainstorming estruturado
- `dispatching-parallel-agents/` - Disparo de agentes paralelos
- `executing-plans/` - Execução de planos
- `finishing-a-development-branch/` - Finalização de branch
- `receiving-code-review/` - Receber code review
- `requesting-code-review/` - Solicitar code review
- `subagent-driven-development/` - Desenvolvimento orientado por subagentes
- `systematic-debugging/` - Debug sistemático
- `test-driven-development/` - TDD
- `using-git-worktrees/` - Uso de git worktrees
- `using-superpowers/` - Como usar o framework
- `verification-before-completion/` - Verificação antes de completar
- `writing-plans/` - Escrita de planos
- `writing-skills/` - Criação de skills
