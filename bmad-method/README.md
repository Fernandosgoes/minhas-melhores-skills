# BMAD METHOD

**Criador**: [bmad-code-org](https://github.com/bmad-code-org)
**Repositório Original**: https://github.com/bmad-code-org/BMAD-METHOD

## Descrição

Framework de metodologia de desenvolvimento com skills para análise, solutioning, planejamento e implementação.

## Estrutura de Skills

### Análise
- `1-analysis/` - Research, Product Brief, PRFAQ, Document Project, Agent Analyst, Agent Tech Writer

### Planejamento
- `2-plan-workflows/` - Create PRD, UX Design, Edit PRD, Validate PRD, Agent PM, Agent UX Designer, Agent Architect

### Solutioning
- `3-solutioning/` - Generate Project Context, Create Epics/Stories, Create Architecture, Check Implementation Readiness

### Implementação
- `4-implementation/` - Agent Dev, Code Review, Checkpoint Preview, Create Story, Dev Story, QA Generate E2E, Quick Dev, Retrospective, Sprint Planning, Sprint Status, Correct Course

### Core Skills
- `core-skills/` - Brainstorming, Advanced Elicitation, Distillator, Editorial Review (Prose/Structure), Help, Index Docs, Party Mode, Review (Adversarial General/Edge Case Hunter), Shard Doc
